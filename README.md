# Codewars Solutions 🥋

Solutions to selected Codewars challenges written in Python.

## Challenges Completed

- **Even or Odd** - Returns whether a number is even or odd
- **Convert a Number to a String** - Converts a number to its string representation
- **Remove String Spaces** - Removes all spaces from a string
- **Vowel Count** - Counts the number of vowels in a string

## Files

- `codewars_solutions.py` - Contains all four solutions

## How to Run

1. Make sure you have Python installed
2. Clone the repository:
   bash
   git clone https://github.com/Rod-ando-le/codewars-solutions.git

3. Navigate into the folder:
   bash
   cd codewars-solutions

4. Run the file:
   bash
   python codewars_solutions.py
